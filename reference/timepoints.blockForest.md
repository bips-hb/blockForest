# blockForest timepoints

Extract unique death times of blockForest Survival forest

## Usage

``` r
# S3 method for class 'blockForest'
timepoints(x, ...)
```

## Arguments

- x:

  blockForest Survival forest object.

- ...:

  Further arguments passed to or from other methods.

## Value

Unique death times
